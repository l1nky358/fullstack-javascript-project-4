Page Loader — утилита командной строки для скачивания веб-страниц с локальными ресурсами (изображениями).  

# Скачать страницу в текущую директорию
page-loader https://ru.hexlet.io/courses

# Скачать страницу в указанную директорию
page-loader --output /var/tmp https://ru.hexlet.io/courses

# Показать справку
page-loader --help
